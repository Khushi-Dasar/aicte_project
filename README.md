# 📄 AI Resume Screening & Candidate Ranking System

## 📌 Overview

This project automates resume screening using Machine Learning and Natural Language Processing (NLP). It ranks resumes based on their similarity to a given job description, helping recruiters quickly identify the most relevant candidates.

---

## 🎯 Objectives

* Automate resume screening
* Reduce manual effort in hiring
* Improve candidate matching accuracy
* Provide unbiased, data-driven ranking

---

## ⚙️ Technologies Used

* Python
* Streamlit (Frontend UI)
* Scikit-learn (TF-IDF & Cosine Similarity)
* PyPDF2 (PDF text extraction)
* Pandas (Data handling)

---

## 🧠 How It Works

1. Upload resumes (PDF format)
2. Enter job description
3. System extracts text from resumes
4. Converts text to TF-IDF vectors
5. Computes cosine similarity
6. Ranks resumes based on relevance score

---

## 📂 Project Structure

```text
resume-ranking-system/
│
├── resumeranking.py
├── requirements.txt
└── README.md
```

---

## 🚀 How to Run

### 🔹 Install dependencies

```bash
pip install streamlit scikit-learn PyPDF2 pandas
```

### 🔹 Run the app

```bash
streamlit run resumeranking.py
```

---

## 📊 Output

* Ranked list of resumes
* Similarity score for each candidate

---

## 🔮 Future Improvements

* Support DOCX & TXT formats
* Add skill extraction using NLP
* Visualize candidate ranking
* Deploy on Streamlit Cloud
